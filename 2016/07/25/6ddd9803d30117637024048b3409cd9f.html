case调用的其实是when后对象的===方法

所谓x === y，就相当于在问y是否属于x所代表的的一类、一组东西，Class#===会返回“是否该类的对象”（注意，反过来写就不对了）：

String === "hello" # true
String === 1 # false


其实String、Regexp、Range都有定义实例方法===，所以你在when后可使用它们的对象来做匹配

Range：

case 5
when (1..10)
  puts "case statements match inclusion in a range"
end


Regexp：

case "FOOBAR"
when /BAR$/
  puts "they can match regular expressions!"
end


proc和lambda：

case 40
when -> (n) { n.to_s == "40" }
  puts "lambdas!"
end


自定义===：

class Success
  def self.===(item)
    item.status >= 200 && item.status < 300
  end
end

class Empty
  def self.===(item)
    item.response_size == 0
  end
end

case http_response
when Empty
  puts "response was empty"
when Success
  puts "response was a success"
end
